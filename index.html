Frequently Asked Questions

Everything you need to know about LinkProtect.

Is my data sent to a server? +
No. Everything happens in your browser. Links are encoded locally and stored only inside the URL.
Will it work on Blogger? +
Yes. Just paste this single HTML file into a Blogger page and it works without any backend.
Are there URL length limits? +
Browsers support very long URLs (8KB+). For huge lists, use an external URL shortener after generating.
Can I monetize the unlock page? +
Yes. Add your ad code (AdSense, Adsterra, etc.) inside the placeholders on the unlock screen.
Is the encryption unbreakable? +
It's obfuscation, not military encryption. It prevents casual scraping and direct link extraction.
